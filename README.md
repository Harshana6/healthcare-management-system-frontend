# Healthcare Management System

A responsive healthcare management website developed during a 3-month internship. The system provides separate interfaces for Admin, Doctor, Receptionist, and Patient users.

## Features

* Admin Dashboard
* Doctor Dashboard
* Reception Dashboard
* Patient Portal
* Interactive Charts and Analytics
* Data Tables and Records Management
* Responsive User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

## Internship Project

Developed as part of a 3-month internship experience focusing on healthcare management workflows, dashboard development, responsive user interfaces, and data visualization.
